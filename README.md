# vue3-first

## 前端项目启动
```
git clone git@github.com:gaecoli/vue3-happy.git
cd vue3-happy
npm install
npm run serve
```

## node 项目启动
### sql 导入：
```
create database ego;

mysql -u root -p ego < vue3-happy/server/ego.sql

```

### node 后端项目启动
```
cd vue3-happy
cd server
node index.js
```
